## 🐍 Useful Python Resources

A curated list of awesome Python frameworks, libraries, software and resources.

---

### 🏛️ Official Sites

*   [Python Programming Language](https://www.python.org/) - Official website.
*   [Enthought](https://www.enthought.com/) - Python tools, training and consulting.
*   [Python Software Foundation (PSF)](https://www.python.org/psf/) - The non-profit organization behind Python.

### ✍️ Blogs

*   [Real Python](https://realpython.com/) - High-quality Python tutorials for all skill levels.
*   [Al Sweigart (CoffeeGhost)](http://coffeeghost.net/) - Blog of the PyGame developer and author of "Automate the Boring Stuff".
*   [Sebastian Raschka](https://sebastianraschka.com/) - Deep dives into Machine Learning and Python.
*   [Ned Batchelder](https://nedbatchelder.com/blog/) - Great insights on Python internals and best practices.
*   [Mouse vs. Python](https://www.blog.pythonlibrary.org/) - Interviews and tutorials by Mike Driscoll.
*   [M.L. Hetland](https://folk.idi.ntnu.no/mlh/hetland_org/) - Prof. at NTNU, author of popular Python books.

### ✉️ Newsletters

*   [Python Weekly](https://www.pythonweekly.com/) - A free weekly roundup of Python news, articles, and jobs.
*   [PyCoder's Weekly](https://pycoders.com/) - Weekly newsletter for those who write Python.
*   [Import Python](https://importpython.com/newsletter/) - Python newsletter with a focus on projects and news.
*   [Python Central](https://www.pythoncentral.io/) - Tips, tricks, recipes and more.

### 🧪 Scientific & Data Science

*   [NumPy](https://numpy.org/) - The fundamental package for scientific computing with Python.
*   [SciPy](https://scipy.org/) - Ecosystem of open-source software for mathematics, science, and engineering.
*   [Matplotlib](https://matplotlib.org/) - A comprehensive library for creating static, animated, and interactive visualizations.
*   [Pandas](https://pandas.pydata.org/) - Powerful data structures for data analysis, time series, and statistics.
*   [SymPy](https://www.sympy.org/) - Python library for symbolic mathematics.
*   [SciKit-Learn](https://scikit-learn.org/) - Simple and efficient tools for predictive data analysis.
*   [PyTorch](https://pytorch.org/) - An open-source machine learning framework.
*   [TensorFlow](https://www.tensorflow.org/) - An end-to-end open source platform for machine learning.
*   [NetworkX](https://networkx.org/) - High-productivity software for complex networks.
*   [Orange](https://orangedatamining.com/) - Open source machine learning and data visualization.

### 🌐 Web Frameworks (WWW)

*   [FastAPI](https://fastapi.tiangolo.com/) - Modern, high-performance, web framework for building APIs with Python 3.8+ based on standard Python type hints.
*   [Django](https://www.djangoproject.com/) - The Web framework for perfectionists with deadlines.
*   [Flask](https://flask.palletsprojects.com/) - A lightweight WSGI web application framework.
*   [Falcon](https://falconframework.org/) - High-performance Python framework for building cloud APIs and microservices.
*   [Tornado](https://www.tornadoweb.org/) - A Python web framework and asynchronous networking library.
*   [Pyramid](https://trypyramid.com/) - A small, fast, down-to-earth Python web framework.
*   [Bottle](https://bottlepy.org/) - Fast, simple and lightweight WSGI micro web-framework.
*   [Sanic](https://sanic.dev/) - Next-generation Python web framework built on top of uvloop and designed for speed.

### 🎓 Tutorials & Workshops

*   [LearnPython.org](https://www.learnpython.org/) - Free interactive Python tutorial.
*   [Full Stack Python](https://www.fullstackpython.com/) - Explains each piece of the web stack in plain language.
*   [Python for Beginners](https://www.pythonforbeginners.com/) - A great place to start for newcomers.
*   [Google's Python Class](https://developers.google.com/edu/python) - A free class for people with a little bit of programming experience.
*   [The Flask Mega-Tutorial](https://blog.miguelgrinberg.com/post/the-flask-mega-tutorial-part-i-hello-world) - An in-depth series for learning Flask.

*   [Python Challenge](http://www.pythonchallenge.com/) - The first web-based programming riddle.
*   [Exercism: Python Track](https://exercism.org/tracks/python) - 100+ Python exercises to help you master the language.

### 🕷️ Scraping

*   [Scrapy](https://scrapy.org/) - An open-source and collaborative framework for extracting data from websites.
*   [Beautiful Soup](https://www.crummy.com/software/BeautifulSoup/) - A library for pulling data out of HTML and XML files.
*   [Playwright for Python](https://playwright.dev/python/) - Reliable end-to-end testing and automation for modern web apps.
*   [HTTPX](https://www.python-httpx.org/) - A next-generation HTTP client for Python.
